### ðŸ›’ 1. Customer Shopping Behavior Analysis  
**Goal:** Analyze shopping data from **3,900 transactions** to understand customer spending patterns, product preferences, and subscription behavior.  

**Tools:** Python (`pandas`, `numpy`), MySQL, Power BI  

**Key Steps:**
- Cleaned and preprocessed transactional data using Python.
